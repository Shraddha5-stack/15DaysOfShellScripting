# Day-14 Server Health Monitoring

## 📌 Objective

The objective of this project is to monitor the health and performance of a Linux server using Bash scripts. These scripts help administrators quickly check CPU usage, memory usage, disk space, system uptime, running processes, and generate a complete server health report.

## 🛠 Scripts

### 1. cpu_usage.sh

**Purpose:**
Displays the current CPU usage.

**Command:**

```bash
./cpu_usage.sh
```

---

### 2. memory_usage.sh

**Purpose:**
Displays RAM and swap memory usage.

**Command:**

```bash
./memory_usage.sh
```

---

### 3. disk_usage.sh

**Purpose:**
Displays filesystem disk usage.

**Command:**

```bash
./disk_usage.sh
```

---

### 4. uptime_check.sh

**Purpose:**
Shows the system uptime and load average.

**Command:**

```bash
./uptime_check.sh
```

---

### 5. process_monitor.sh

**Purpose:**
Displays the top CPU-consuming processes.

**Command:**

```bash
./process_monitor.sh
```

---

### 6. server_health_report.sh

**Purpose:**
Generates a complete server health report.

**Command:**

```bash
./server_health_report.sh
```
