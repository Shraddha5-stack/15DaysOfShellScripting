# Day 02 - Variables & Operators Assignment

## Task 1: Create Variables

Create a shell script that stores:

- Name
- City
- Course
- Experience

Example:

```bash
NAME="Shraddha"
CITY="Pune"
COURSE="Shell Scripting"
EXPERIENCE="Beginner"
```

Display all values using echo.

---

## Task 2: Environment Variables

Display the following Linux environment variables:

```bash
$HOME
$USER
$PWD
$SHELL
$PATH
```

---

## Task 3: User Input

Create a script that accepts:

- User name
- City
- Course

using the `read` command.

Example:

```bash
read NAME
read CITY
read COURSE
```

---

## Task 4: Arithmetic Operations

Create variables:
