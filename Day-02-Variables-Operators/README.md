# Day 2 – Variables, Operators & User Input

## 📌 Overview

Welcome to Day 2 of my **15 Days of Shell Scripting Challenge**.

In Day 1, I learned the fundamentals of Bash scripting, including the shell, shebang, comments, and variables.

In Day 2, I focused on making shell scripts interactive by learning how to use variables, take user input, and perform different types of operations.

---

## 🎯 Learning Objectives

By the end of Day 2, I will be able to:

- Understand variables in Bash
- Use environment variables
- Accept user input using the `read` command
- Perform arithmetic operations
- Understand relational and logical operators
- Check command exit status using `$?`
- Create interactive shell scripts

---

## 📚 Topics Covered

- Variables
- Environment Variables
- User Input (`read`)
- Arithmetic Operators
- Relational Operators
- Logical Operators
- Assignment Operators
- Exit Status (`$?`)
