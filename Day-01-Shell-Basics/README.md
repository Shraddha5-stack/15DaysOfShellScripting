# 🐚 Day 1 - Shell Basics

## 📖 Introduction

Welcome to **Day 1** of the **15 Days of Shell Scripting Challenge**.

Today focuses on understanding the fundamentals of Shell Scripting. Before writing automation scripts, it's important to understand what a shell is, how it works, and why shell scripting is widely used in Linux and DevOps.

This day builds the foundation for the rest of the challenge.

---

## 🎯 Learning Objectives

By the end of Day 1, you will be able to:

- Understand what Linux and UNIX are.
- Explain what a Shell is.
- Identify different types of Shells.
- Understand Bash and why it is commonly used.
- Explain what Shell Scripting is.
- Understand the purpose of the Shebang (`#!/bin/bash`).
- Execute shell scripts using different methods.
- Work with variables and comments.
- Write and execute your first shell script.
