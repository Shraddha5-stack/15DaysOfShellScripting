## Task 1

Create a script named:

```text

```

Requirements:

- Add a shebang.
- Print a welcome message.
- Print your name.
- Print today's date.

## Task 2

Create a script named:

```text

```

- Current user
- Current working directory
- Hostname

## Task 3
