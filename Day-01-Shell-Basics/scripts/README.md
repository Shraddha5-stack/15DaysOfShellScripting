# Day 1 - Shell Scripts

This directory contains the practical shell scripts created during **Day 1** of the **15 Days of Shell Scripting Challenge**.

## Scripts

| Script | Description |
|---------|-------------|
| hello.sh | Prints a welcome message |
| comments.sh | Demonstrates single-line comments |
| variables.sh | Demonstrates user-defined variables |
| system-info.sh | Displays basic system information |
| executable-script.sh | Demonstrates script execution |

## Skills Learned

- Writing basic shell scripts
- Using the shebang (`#!/bin/bash`)
- Displaying output with `echo`
- Working with variables
- Running scripts with `./script.sh`
- Making scripts executable using `chmod +x`
