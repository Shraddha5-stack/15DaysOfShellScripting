# Day 05 - Loops in Bash Scripting

## 📌 Overview

Loops are one of the most important concepts in Bash scripting. They allow us to execute a block of code multiple times without writing the same code repeatedly. Loops help automate repetitive tasks, making scripts shorter, cleaner, and more efficient.

In this day, I learned different types of loops in Bash, loop control statements, and built practical scripts using loops.

---

## 🎯 Learning Objectives

- Understand the purpose of loops in Bash.
- Learn the `for`, `while`, and `until` loops.
- Understand the difference between `break` and `continue`.
- Practice using loops with variables and conditions.
- Build real-world Bash scripting examples.
