## 📖 About This Repository

This repository documents my **15-Day Shell Scripting Challenge**, where I am learning Shell Scripting from the ground up.

The goal is to build a strong foundation in:

- Linux Fundamentals
- Shell Scripting
- Linux Commands
- Bash Programming
- Automation
- DevOps Concepts
- Real-world Projects

Each day contains:

- 📘 Theory Notes
- 💻 Practical Scripts
- 📋 Linux Commands
- ❓ Interview Questions & Answers
- 🧪 Practice Exercises
- 📝 Assignments
- 📸 Output Screenshots
- 🚀 Mini Projects
