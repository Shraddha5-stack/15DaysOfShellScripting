# Day-13 User Management Project

## 📌 Objective

The objective of this project is to learn Linux user management using Bash scripting. This project demonstrates how to automate common administrative tasks such as creating users, deleting users, listing users, locking/unlocking accounts, and displaying user information.

## 🛠 Scripts

### 1. create_user.sh

**Purpose:**
Creates a new Linux user account.

**Command:**

```bash
./create_user.sh
```

---

### 2. delete_user.sh

**Purpose:**
Deletes an existing Linux user account.

**Command:**

```bash
./delete_user.sh
```

---

### 3. list_users.sh

**Purpose:**
Lists all users available on the system.

**Command:**

```bash
./list_users.sh
```

---

### 4. lock_unlock_user.sh

**Purpose:**
Locks or unlocks a Linux user account.

**Command:**

```bash
./lock_unlock_user.sh
```

---

### 5. user_info.sh

**Purpose:**
Displays detailed information about a Linux user.

**Command:**

```bash
./user_info.sh
```
