# Day 04 – Conditional Statements in Bash

## 📌 Objective

The objective of Day 04 is to understand how Bash scripts make decisions using conditional statements. Conditional statements allow a script to execute different blocks of code based on whether a condition is true or false.

These concepts are widely used in Linux administration, automation, and DevOps.

---

# 📚 Topics Covered

- if statement
- if-else statement
- if-elif-else ladder
- Nested if statement
- case statement
- Numeric comparison operators
- String comparison operators
- File test operators
- Logical operators
- Real-world decision-making scripts
