# Day 04 – Bash Scripts

This folder contains practical Bash scripts demonstrating conditional statements.

## Scripts Included

### 1. if-statement.sh

Checks whether a given number is positive.

### 2. if-else.sh

Checks whether a person is eligible to vote based on age.

### 3. elif-ladder.sh

Calculates student grades using multiple conditions.

### 4. nested-if.sh

Performs username and password verification using nested `if` statements.

### 5. case-statement.sh

Implements a menu-driven calculator using the `case` statement.

## Skills Practiced

- if statement
- if-else
- if-elif-else
- Nested if
- case statement
- Numeric comparisons
- String comparisons
- User input
- Decision making
- Interactive Bash scripting

These scripts demonstrate how conditional logic is used in Linux administration and DevOps automation.
