# Day 04 – Assignment

## Objective

Practice Bash conditional statements by creating small programs using `if`, `if-else`, `if-elif-else`, nested `if`, and `case`.

---

## Task 1 – Even or Odd Number

Write a Bash script that:

- Accepts a number from the user.
- Prints whether the number is Even or Odd.

---

## Task 2 – Largest of Three Numbers

Write a Bash script that:

- Accepts three numbers.
- Displays the largest number.

---

## Task 3 – Leap Year Checker

Write a Bash script that:

- Accepts a year.
- Checks whether it is a leap year.

---

## Task 4 – File Checker

Write a Bash script that:

- Accepts a file name.
- Checks whether the file exists.
- Displays an appropriate message.

---

## Task 5 – Simple Menu

Create a menu using the `case` statement.

Menu:

1. Show Current Date
2. Show Current Directory
3. Show Logged-in User
4. Exit

Execute the selected option.

---

## Bonus Challenge

Create a login system that:

- Accepts username and password.
- Gives the user only **3 attempts**.
- Displays "Access Granted" or "Access Denied".

---

## Learning Outcome

After completing this assignment, I can:

- Write decision-making scripts.
- Use conditional statements confidently.
- Build interactive Bash programs.
- Apply conditional logic to real-world Linux tasks.
