# Day 06 - Bash Arrays

## 📌 Overview

On Day 06, I learned how to work with arrays in Bash Shell Scripting. Arrays allow us to store multiple values in a single variable, making scripts easier to manage and more efficient.

---

## 🎯 Topics Covered

* Creating arrays
* Accessing array elements
* Displaying all array elements
* Finding the length of an array
* Looping through arrays
* Adding new elements
* Updating existing elements
* Removing elements using `unset`

## 📜 Scripts

### 1. array_basics.sh

Demonstrates the basics of Bash arrays, including:

* Creating an array
* Accessing elements
* Printing all elements
* Finding array length
* Looping through an array
* Adding, updating, and removing elements

### 2. student_names.sh

Stores student names in an array and displays each name using a `for` loop.

### 3. programming_languages.sh

Stores programming language names in an array, prints all languages, counts them, and displays each one individually.
